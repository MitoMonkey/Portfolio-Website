MoveX

MoveX is a library of movements from various styles and traditions, with descriptions/cues and video sources.

For this study project I built a REST API and a non-relational database as the backend, as well as a React-based frontend client

Main technologies used for the client: Javascript, React, React Bootstrap, React Redux, Axios, Parcel, PropTypes, hosting on Netlify

Main technologies used for the backend: Javascript, non-relational DB on MongoDB Atlas, App hosted on Heroku, Postman (API testing), node.js (Express, Morgan, bodyParser, mongoose, CORS, Passport, bcrypt)

Live version See client on GitHub See backend on GitHub
